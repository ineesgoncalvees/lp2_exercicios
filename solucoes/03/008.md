## Problema

8 - Considera as interfaces
[`IList<T>`](https://docs.microsoft.com/dotnet/api/system.collections.generic.ilist-1)
e
[`IDictionary<TKey,TValue>`](https://docs.microsoft.com/dotnet/api/system.collections.generic.idictionary-2)
.

1.  Para cada uma das interfaces, dá um exemplo de uma coleção do C# que a
    implemente.
2.  Qual é ou quais são as funcionalidades que as coleções que implementam
    estas interfaces são obrigadas a ter?

## Soluções

### Solução 1

Um exemplo de uma coleção que implementa a interface `IList<T>` é `ArrayList`.
As coleções que implementam esta interface são obrigadas a ter as seguintes
funcionalidades:

* `Item[Int32], recebe ou devolve o elemento a um indice especifico;

* `IndexOf(T), determina o indice de um item especifico na lista;
* `Insert(Int32, T), insere um item à lista no indice especificado;
* `RemoveAt(Int32), remove o item de um indice especifico.

Um exemplo de uma coleção que implementa a interface `IDictionary<TKey,TValue>`
é .
As coleções que implementam esta interface são obrigadas a ter as seguintes
funcionalidades:

* `Item[TKey]`, recebe ou devolve o elemento a uma chave especifica;
* `Keys`, recebe uma coleção que contem chaves do dicionario;
* `Values`, recebe uma coleção que contem chaves no dicionario.

* `Add(TKey, TValue)`, adiciona um elemento com o chave e o valor dados ao 
dicionario;
* `ContainsKey(TKey)`, determina se o dicionario contem ou não um elementeo com
uma chave especifica;
* `Remove(TKey)`, remove o elemento com a chave espeficica do dicionario;
* `TryGetValue(TKey, TValue)`, devolve o valor associado com a chave especificada.
